# EasySharp
 
A C# internal base for CSGO with an implemented skinchanger.
 
![EasySharp](https://image.noelshack.com/fichiers/2020/19/6/1589053385-7.jpg)

## Installation

1. Go in Visual Studio, in Nuget Package Manager, in Manage Nuget Packages for Solution
2. Search for EasyHook and install the Nuget for SharpSkin and SharpSkin_Injector.
3. Compile
4. Put in the same folder :
  - EasyHook.dll
  - EasyLoad32.dll
  - EasyHook32.dll
  - EasySharp.dll
  - EasySharp_Injector.exe
5. Run the injector
